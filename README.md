## Get started

1. Clone the repository & install dependencies

   ```bash
   npm install
   ```

3. Add the .env file and paste the required EXPO_PUBLIC variables

   ```bash
   code .env
   ```

2. Start the app

   ```bash
   npx expo start -c
   ```
